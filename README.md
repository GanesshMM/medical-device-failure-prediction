
# MedStream - Medical Device Failure Prediction (Project Package)

This package contains code, models, and documentation for the MedStream predictive maintenance demo.

## Files included
- index.html, app.js, style.css: Frontend dashboard files
- predict_server.py: FastAPI backend that serves predictions
- gbc_best.pkl, gbr_best.pkl, xgb_best.pkl, xgbr_best.pkl, label_encoder.pkl: Model artifacts (if present)
- Medical_Device_Failure_Prediction_Report.pdf: Short autogenerated report
- Full_Project_Documentation.md: Detailed documentation to paste into final report

## Quick Start
1. Install dependencies:
   pip install -r requirements.txt
2. Start backend:
   uvicorn predict_server:app --reload --port 8000
3. Serve frontend or open index.html in a browser:
   python -m http.server 8080
4. Open frontend at http://localhost:8080/index.html
